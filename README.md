# 🏙️ Local Community Problem Solver (LCPS)

A full-stack civic complaint management system developed using Java Servlets, JSP, JDBC, and MySQL following the MVC2 architecture.

LCPS allows citizens to report local civic issues, authorities to review complaints, workers to resolve issues, and administrators to monitor the entire system through dashboards and analytics.

---

# 🚀 Features

## Citizen Module
- Register and Login with OTP Verification
- Submit Civic Complaints
- Upload Complaint Images
- Track Complaint Status
- View Resolution Timeline

## Authority Module
- Department-wise Complaint Management
- Review Complaints
- Assign Workers
- Approve or Reject Completed Work
- Monitor Department Performance

## Worker Module
- View Assigned Complaints
- Update Progress
- Upload After-Work Images
- Mark Complaints as Completed

## Admin Module
- Manage Users
- Manage Departments
- View System Analytics
- View Complaint Statistics
- Reassign Complaints
